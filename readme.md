## **Особенности обучения:**

* учат онлайн из любой точки мира;
* CSS, HTML, Java Script и другие важные навыки;
* Практика занимает ~~450 часов из 560~~ все ваше свободное время!
* После обучения помогают трудоустроиться;
* 12 проектов в портфолио: сайты и веб-приложения. Данный проект - это один из них.


В данном проекте использованы: HTML5, CSS, БЭМ, а также знания, усилия и помощь сотен людей.
